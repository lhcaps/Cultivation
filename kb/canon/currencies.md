# Currencies — Thiên Nam Engine

> The five currencies of Thiên Nam and how they flow.

---

## The Five Currencies

### Bạc (Silver)

The basic currency. Earned from daily rewards, cultivation, quests, and combat. Spent on travel, repairs, and basic items.

```
Acquisition: Daily + cultivation + quests + combat
Sink: Travel + repair + storage + auction fee
```

### Linh Thạch (Spirit Stone)

The advancement currency. Earned from breakthrough, alchemy, and rare drops. Spent on manuals, pills, and advancement.

```
Acquisition: Breakthrough + alchemy + boss kills + rare encounters
Sink: Manuals + pills + advancement + resurrection
```

### Công Huân (Merit)

The sect currency. Earned from sect activities. Spent on sect shop and rank benefits.

```
Acquisition: Sect missions + defense + kills + donations
Sink: Sect shop + rank advancement
```

### Danh Vọng (Reputation)

The community currency. Earned from public deeds. Spent on reputation-only items and faction access.

```
Acquisition: Quests + public events + sect activities
Sink: Reputation items + faction rank
```

### Thiên Đạo Ấn (Heaven's Seal)

The event currency. Earned from world events and season rewards. Spent on exclusive seasonal items and titles.

```
Acquisition: World bosses + season rewards + admin grants
Sink: Seasonal items + exclusive titles
```

---

## Flow Example

```
Daily cultivation → Silver + Linh Thạch (chance)
↓
Sect mission → Merit + Silver
↓
Breakthrough → Linh Thạch
↓
Alchemy → Pills (spend Linh Thạch, earn pills)
↓
Quest → Danh Vọng + Silver
↓
World event → Thiên Đạo Ấn (rare)
```
