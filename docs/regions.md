## 3. Travel System

### 3.1 Travel Costs

| Route Type | Cost (Silver) | Time |
|------------|-------------|------|
| Intra-region | 10-50 Ag | Instant |
| Adjacent region | 100-200 Ag | 1 hour |
| Cross-continent | 300-500 Ag | 4 hours |
| Remote (Côn Lôn ↔ Nam Man) | 500 Ag | 8 hours |

```
Đại Việt ↔ Trung Nguyên, Đông Hải, Nam Man
```
