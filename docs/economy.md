## 6. Economy Health Metrics

### 6.1 Key Indicators

| Metric | Formula | Healthy Range | Warning | Critical |
|--------|---------|--------------|---------|---------|
| **Silver per player/day** | Total Ag ÷ Active players | 500-3000 | >5000 | >20000 |
| **ST circulation/week** | Total ST generated | 50-200/week | >500 | >2000 |
| **Basic pill price** | Market rate | 15-30 ST | 30-100 ST | >100 ST |
| **Equipment price** | Market rate | Baseline ±20% | ±50% | ±100% |
| **Sink/Source ratio** | Sinks ÷ Sources | 0.8-1.2 | 0.5-0.8 | <0.5 or >1.5 |

### 6.2 Economy Intervention Triggers

| Trigger | Condition | Auto-action | Admin Alert |
|---------|-----------|------------|-------------|
| Inflation detected | Ag/player > 10000 | Reduce quest rewards 5% | Level 2+ |
| Deflation detected | Ag/player < 200 | Increase daily reward 10% | Level 2+ |
| Item scarcity | Pill price >200% baseline | Reduce drop rate adjustment | Level 3+ |
| Item abundance | Pill price <50% baseline | Reduce spawn 10% | Level 3+ |
| Merit imbalance | Top sect 10x treasury | Suggest tax increase | Level 2+ |

## 7. Auction House

### 7.1 Listing Rules

| Rule | Limit |
|------|-------|
| Listing fee | 5% of item value (min 10 Ag) |
| Maximum listings | 10 per player |
| Listing duration | 1, 3, or 7 days |
| Automatic relist | Optional, fee applies |
| Rescinded listing | No refund of fee |

### 7.2 Auction Categories

| Category | Item Types | Notes |
|----------|-----------|-------|
| Equipment | Weapons, armor, accessories | Sorted by tier |
| Pills | All consumable pills | Sorted by quality |
| Materials | Herbs, ores, components | Sorted by rarity |
| Manuals | All learning materials | Sorted by realm |
| Misc | All other items | Open category |

### 7.3 Bidding Rules

| Rule | Value |
|------|-------|
| Minimum bid increment | 10% above current bid |
| Bid extension | +5 minutes if bid in last 5 minutes |
| Outbid notification | Automatic DM |
| Failed payment | Item relisted, bidder flagged |
| Winner notification | Automatic DM |

## 8. Taxation

### 8.1 Tax Rates

| Tax Type | Rate | Target | Purpose |
|----------|------|--------|---------|
| **Travel tax** | 10% | All travel purchases | Regional maintenance |
| **Auction tax** | 5% | Auction house sales | Server maintenance |
| **Sect contribution** | 10% | Member earnings to sect | Sect treasury |
| **Trade tax** | 2% | Player-to-player silver | Economy balance |
| **Death tax** | 5% | Revival cost | Prevent exploitation |

### 8.2 Tax Distribution

```
Total Tax Collected = X

Distribution:
├── Server Fund (40%): Operating costs
├── Event Fund (30%): World events, rewards
├── Prize Pool (20%): Leaderboard, competitions
└── Reserve (10%): Emergency interventions
```
